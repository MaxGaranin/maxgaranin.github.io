# Видеотека. Финальный проект по курсу обучения "Разработчик JavaScript" в OTUS

Так как я не фронтэнд-разработчик и это мой первый опыт, то я решил сделать проект не с нуля, а имея перед глазами похожий пример.
Проект сделан на основе статьи [JavaScript 2016, а можно попроще?](https://habr.com/post/313234/), где для реализации использовался React.
Мне было интересно изучить возможности фреймворка Angular, поэтому я использовал его.
